## Тема 9(вариативная часть)

### Информационная среда школы №490. 

#### Достоинства информационной среды:

- Наличие электронного журнала
- Наличие электронного дневника, к которому имеют доступ как сам ученик, так и его родители.
- Наличие компьютерных классов
- Наличие собственной сети Wi-Fi
- Наличие цензора на компьютерах, который не позволяет учащимся заходить на нежелательные сайты.



#### Недостатки информационной среды:

- Некачетсвенный и неудобный сайт школы
- Отсутвие дистанционного обучения
- Устаревшие компьютеры, которые не позволяют в полной мере освоить дисциплину, связанную с их использованием
- Малое количетво педагогов, умеющих взаимодействовать с современной информационной средой
- Неудобный электронный дневник, который к тому же большую часть времени не работает



#### Предложений по совершенствованию информационной среды школы:

- Необходимо обновить сайт школы и сделать его максимально удобным для обычного пользователя, а конкретно для родителей и детей. Также необходимо обновить всю информацию, представленную информацию, так как многое уже давно не соответсвует действительности. (решить данную проблему можно при помощи учащихся даннной школы, дав им задание придумать идею для оформления сайта школы)
- Необходимо обновить материально-техническую базу (обновить компьютеры)
- Необходимо решить проблему с электронным дневником, так как большую часть времени он не работает (нанять сетевого администратора)
- Необходимо ввести в использование систему управления обучением на подобие Moodle для решения проблемы дистанционного обучения
- Необходимо более активно отправлять педагогов на курсы по повышению информационной грамотности.
